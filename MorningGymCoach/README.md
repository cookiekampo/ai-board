# Morning Gym Coach (v0.1)

## アプリの目的
朝ジムでスマホ脱線を防ぎ、30〜45分の筋トレを1回分完走するための個人用iPhoneアプリです。決められたメニューを順番に実施し、セット結果を最小入力で記録します。

## v0.1の機能
- 固定テンプレメニュー（胸・背中 / 45分）
- 今日のメニュー表示
- セット結果入力（重量・回数・RIR）
- RIR 5択（4+, 3, 2, 1, 0）
- RIR→RPE自動変換
- 予定データと実績データを分離して保存
- セット記録後に自動で休憩タイマー開始
- タイマー画面の最小表示（残り秒数 / 次重量 / 目標回数 / 注意文）
- 次セット指示（最大3行）
- 種目タイプ対応（heavy_compound / machine / cable / isolation / bodyweight）
- heavy_compoundでRIR0を禁止
- 最後の安全種目のみRIR0許可
- 痛みボタン（種目終了 + 当日オールアウト禁止）
- 直前セットの修正
- 直前セットの削除
- ローカル保存（UserDefaults）
- トレ後まとめ表示
- 予定と実績の差分表示

## v0.1でやらないこと

## 初期メニュー
### 胸・背中 / 45分
1. チェストプレス（machine）
   - 50kg, 3セット, 8-10回, target RIR [2,1,1], 120秒休憩
2. ラットプル（machine）
   - 45kg, 3セット, 8-12回, target RIR [2,1,1], 120秒休憩
3. ケーブルフライ（cable）
   - 15kg, 2セット, 12-15回, target RIR [1,0], 60秒休憩
   - all_out_allowed=true, rest_pause_allowed=true

## RIR/RPEルール
- RIR 4+ → RPE 6
- RIR 3 → RPE 7
- RIR 2 → RPE 8
- RIR 1 → RPE 9
- RIR 0 → RPE 10

次セット判断:
- RIR4+: 同重量 / 次回アップ候補 / 90秒
- RIR3: 同重量 / 90秒
- RIR2: 同重量 / 120秒
- RIR1: 同重量 + 目標回数を少し下げる / 150秒
- RIR0: 原則種目終了（最後の安全種目のみ許可）
